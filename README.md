
# Twitter-Clone Backend

This is a backend project twitter-clone.

## Installation

Use the package manager [npm](https://pip.pypa.io/en/stable/) to install the dependencies.

```bash
npm install
```

## Usage

```JavaSacript

Initialize the backend run:

npm run start

```
